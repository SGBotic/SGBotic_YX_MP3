## Arduino library for MP3 Module ([DEV-02731](http://www.sgbotic.com/index.php?dispatch=products.view&product_id=2731))

## Wiring 
### Software Serial 
   **MP3 module   ------> Arduino Uno**   
         Rx       ------> Pin 11 (Tx)
         Tx       ------> Pin 10 (Rx)
         Vcc      ------> 3.3V or 5V
         GND      ------> GND

### **Hardware Serial (Mega, Micro, Leonardo, etc)**  
   **MP3 module   ------> Arduino Mega (TX3)**  
         Rx       ------> Pin 14 (Tx3)
         Tx       ------> Pin 15 (Rx3)
         Vcc      ------> 3.3V or 5V
         GND      ------> GND
